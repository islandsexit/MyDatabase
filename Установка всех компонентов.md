Я устанавливал на [[Ubuntu Server]] 

Для начала нужно поставить сам [[Python]] и [[Nginx]]
Также все зависисмые модули
```bash
apt-get install python3-pip python3-dev libpq-dev curl nginx -y
```

1. python3-pip - Питоновский [[Pip]] 
2. [[python3-dev]] 
3. [[libpq-dev]] 
4. [[Curl]] 
5. nginx 
6. [[Флаг -y]]
